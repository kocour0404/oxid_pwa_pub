## 2024-06-17 - XSS in PWA string interpolation\n**Vulnerability:** XSS vulnerability identified in frontend due to unsanitized interpolation of database/user content into DOM via innerHTML.\n**Learning:** In vanilla JS/HTML applications without a framework, injecting data straight into strings requires manual escaping.\n**Prevention:** Create an escapeHtml utility and consistently wrap any interpolated data variables when using innerHTML.
